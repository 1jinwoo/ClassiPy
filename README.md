# ClassiPy
## Features
- Takes search string as input and output category for the product that the search string is aimed to find.
- Best-performing model uses neural network with one layer with 330 neurons

## Dependencies
- NumPy 1.15.4
- Pandas 0.23.4
- TensorFlow 1.12.0
- Keras 2.2.4
- sklearn 0.20.1
- flask 1.0.2
